The Input box component provides styling for most prompts to enter data in forms. While this component is compatible with SKY UX's default theme, it provides more styling in modern theme.

Use input box when users need to enter data in forms. The Input box component provides styling for many types of data entry, from basic single-line text, to text areas, to other SKY UX inputs like Datepicker.
