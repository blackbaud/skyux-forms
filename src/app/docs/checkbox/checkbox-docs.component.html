Checkboxes enable users to select multiple items from a short list of items.

For a group of checkboxes, use fieldset and legend elements to provide semantic structure. For people using assistive technology, the fieldset element groups the checkboxes to indicate the number of items in the group, and the legend element provides a description for the group. For more information on how to use fieldset and legend, see WebAIM examples and guidance.

For icon checkboxes, the label or labelledby input must be set. For people using assistive technology, the visible icon may not convey sufficient information.
